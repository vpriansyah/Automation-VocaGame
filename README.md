# VocaGame Automation Testing

## Overview
This project contains automation testing for the VocaGame website using Katalon Studio. The automation focuses on validating core user flows such as registration, login, forgot password, and product ordering. The goal of this project is to ensure that the main functionalities work correctly and that the system properly handles both positive and negative scenarios.

## Scope of Testing
- User Registration
- User Login
- Forgot Password
- Product Order

## Test Approach
Automation tests simulate user actions such as filling forms, clicking buttons, and submitting requests. Assertions are used to verify that the system behaves as expected and displays the correct error messages when invalid inputs are provided.
